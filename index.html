Profile-box

An easy to implement design for profiles.

Download this project as a .zip file Download this project as a tar.gz file

Do you run a website, and have staff?

Then this is the program for you. It's a simple and easy to use HTML and CSS bundle. All you need to do is link in the profile css, from your folder or github and it is ready to use. Add in the divs and it's fully functional. As an open source html and css file you can change what you need!

Uses

Some simple uses are for websites such as:

  • Business Websites
  • Personal Portfolios
  • Software Website
  • Design Websites
  • Blogs with many staff
